# dustbinweb

##  更新优化方向

1.  批量修改传感器测定周期等参数，实现形式API；

2.  标签管理某一批次产品，实现形式API，用户+标签

## 新框架整理利用面向对象思想

1. 用户类
  基本用户类  可新增下一级用户 可修改测定周期 可修改标记量程 
  代理商用户类 
  用户绑定设备的方式
  1.用户直接绑定
  2.用户分组绑定
  3.用户地区绑定
2. 设备类
  基本设备信息类
  设备地理信息类 真实GPS位置、校正GPS位置、固定GPS位置、展示位置选择、多种方式设置校正GPS位置（GPS角度、GPS经纬度、BD09经纬度、地址）设备分区表、设备区域顺序表、区域坐标排序
  设备设置、问题、回收、提示等信息类
  设备生命周期类 生产 重点监测 放任监测
3. 统计类
  垃圾产生量统计
  设备通信故障率统计
  垃圾溢出量统计
  垃圾回收效率统计
4. 数据监控类
  

5. 数据分析类
